## Topics Covered

### 1. Basics & Variables

- Understanding `let` vs `const`.
- Primitive types (String, Number, Boolean, undefined, null).
- Dynamic typing in JavaScript.

### 2. Operators

- Arithmetic, Assignment, and Comparison operators.
- Equality operators (`==` vs `===`).
- Ternary operators and Logical operators with non-booleans (short-circuiting).

### 3. Control Flow

- Conditional statements (`if...else`, `switch...case`).
- Loops: `for`, `while`, `do...while`, `for...in`, and `for...of`.
- **Exercises:** FizzBuzz, Max of Two Numbers, Demerit Points calculator, etc.

### 4. Objects

- Creating objects using Literals, Factory Functions, and Constructor Functions.
- Dynamic nature of objects and the `this` keyword.
- Value vs. Reference types.

### 5. Arrays

- Adding, removing, and finding elements.
- Arrow functions.
- Powerful array methods: `map`, `filter`, `reduce`, and `sort`.

### 6. Functions

- Function declarations vs. expressions.
- Rest operator and default parameters.
- Getters, setters, and error handling (`try...catch`).
